Clone or copy the structure.

Install dependencies in /server and /client using `npm install`.

Start the server: in /server, run `npm start` (it will listen on port 3001 by default).

Start the client: in /client, run `npm run dev` (Vite). Open http://localhost:5173.

In the client, enter a name and click "Join" to join the main lobby. Open another window to test multiplayer mode.
